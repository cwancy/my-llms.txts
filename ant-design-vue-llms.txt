# Ant Design Vue

> An enterprise-class UI component library for Vue 3, implementing the Ant Design specification.
> Package: `ant-design-vue` | Latest stable: 4.2.6 | License: MIT
> Homepage: https://antdv.com | GitHub: https://github.com/vueComponent/ant-design-vue

## Overview

Ant Design Vue (antdv) is the official Vue implementation of Alibaba's Ant Design system.
It provides ~80 high-quality components for building enterprise desktop web applications.
Supports Vue 3 (v3+/v4+), Server-Side Rendering, and Electron.
Icons are in a separate package: `@ant-design/icons-vue`.

## Installation

```bash
npm install ant-design-vue
# or
yarn add ant-design-vue
```

### Full import (main.ts)

```ts
import { createApp } from 'vue'
import Antd from 'ant-design-vue'
import 'ant-design-vue/dist/reset.css'
import App from './App.vue'

createApp(App).use(Antd).mount('#app')
```

### On-demand import (recommended, via unplugin-vue-components)

```ts
// vite.config.ts
import Components from 'unplugin-vue-components/vite'
import { AntDesignVueResolver } from 'unplugin-vue-components/resolvers'

export default defineConfig({
  plugins: [
    Components({
      resolvers: [AntDesignVueResolver({ importStyle: false })]
    })
  ]
})
```

### Manual tree-shaking

```ts
import { Button, DatePicker } from 'ant-design-vue'
app.use(Button).use(DatePicker)
```

## Component tag prefix

All components use the `a-` prefix in templates.
e.g. `<a-button>`, `<a-table>`, `<a-form-item>`

## Theming (Design Tokens)

Ant Design Vue 4.x uses a CSS-in-JS token system via `ConfigProvider`.

```ts
<a-config-provider
  :theme="{
    token: {
      colorPrimary: '#00b96b',
      borderRadius: 4,
      colorBgContainer: '#f6ffed',
    },
    algorithm: theme.darkAlgorithm, // or theme.compactAlgorithm
  }"
>
```

Import algorithms:
```ts
import { theme } from 'ant-design-vue'
const { defaultAlgorithm, darkAlgorithm, compactAlgorithm } = theme
```

## Form Validation

```ts
import { Form } from 'ant-design-vue'
const { useForm } = Form

const formState = reactive({ username: '', password: '' })
const rules = {
  username: [{ required: true, message: 'Required', trigger: 'blur' }],
  password: [{ min: 6, message: 'Min 6 chars', trigger: 'change' }],
}
const { validate, validateInfos, resetFields } = useForm(formState, rules)

const onSubmit = () => validate().then(() => { /* submit */ })
```
